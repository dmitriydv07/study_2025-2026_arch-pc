# Цель работы
Освоить подготовку отчёта в формате Markdown и сборку отчёта средствами Makefile, а также базовые операции с git.

# Задание
1. Перейти в каталог лабораторной работы №2.
2. Подготовить файл отчёта `report.md` в текстовом редакторе.
3. Собрать отчёт в PDF/DOCX с помощью `make`.
4. Добавить результаты в git-репозиторий и отправить на GitHub.
